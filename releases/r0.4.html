Description

Release 0.4 of the MOTECH platform is meant as an early technology release focused on developers. At this point in time the system is functional, but we have not spent much time yet on documentation or user interface so the use of the system for non-developers is limited.

Release Notes

Please refer to the following link for the release notes for this release.

Build Instructions

You can build the platform by checking out the motech-0.4 tag and running 
mvn install
You can include any platform module in your mvn build by placing the following in your pom: {% highlight xml %} motech-repo Motech Maven Repository http://nexus.motechproject.org/content/repositories/public org.motechproject [module name] 0.4 {% endhighlight %}

Installation Instructions

We will be releasing a puppet script that can be run on Ubuntu to install and start pre-requisite services. The script will also pull down release 0.4 and install it in a tomcat container. If you want to manually install MOTECH you will need:
  • Tomcat
  • MySQL
  • CouchDB
  • Java
  • ActiveMQ

Modules

Name Status Documentation Download Description

Appointments Beta In Progress jar Provides appointment scheduling and reminding
CMS Lite Beta In Progress jar Provides basic content storage and retrieval
Decision Tree Beta In Progress jar Provides APIs for constructing an IVR decision tree
Event Tracking Alpha In Progress jar Provides basic operations for publishing and subscribing events

IVR Voxeo Alpha In Progress jar Integrates the platform with Voxeo's hosted IVR
Message Campaign Beta Description | Events | Questions jar Enrolls users in message campaigns with flexible content scheduling rules
MRS (Medical Record System) Alpha In Progress jar Provides basic specification for integrating platform with a medical record system
OpenMRS Alpha In Progress jar Integrates platform with OpenMRS system
OpenMRS OMOD Alpha In Progress jar Provides OpenMRS specific OMOD implementation
Outbox Beta Description | Events | Questions jar A voicemail like messaging system for end users
Pill Reminder Beta In Progress jar A flexible reminder system focused on medication
Scheduler Beta In Progress jar Provides APIs for scheduling and unscheduling reminder jobs
Schedule Tracking Alpha Description | Events | Questions jar Enrolls users for alerts based on complex scheduling rules
SMS Beta In Progress jar Provides basic specification for integrating platform with an SMS provider to send out SMS messages
