Description

Developer Release 1 of the MOTECH platform is meant as an early technology release focused on developers. At this point in time the system is functional, but we haven't spent much time yet on documentation or user interface so the use of the system for non-developers is limited.

Release Notes

Build Instructions

You can build the platform by checking out the DR1 tag and running 
mvn install
You can include any platform module in your mvn build by placing the following in your pom: {% highlight xml %} motech-repo Motech Maven Repository http://nexus.motechproject.org/content/repositories/public org.motechproject [module name] DR1 {% endhighlight %}

Installation Instructions

We will be releasing a puppet script that can be run on Ubunutu to install and start pre-requisite services. The script will also pull down the DR1 release and install it in a tomcat container. If you want to manually install Motech you will need:
  • Tomcat
  • MySQL
  • CouchDB
  • java
  • ActiveMQ
