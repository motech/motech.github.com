Description

Release 0.12 of the MOTECH platform is meant as an early technology release focused on developers. At this point in time the system is functional, but we have not spent much time yet on documentation or user interface so the use of the system for non-developers is limited.

Release Notes

Please refer to the following link for the release notes for this release.

Build Instructions

You can build the platform by checking out the motech-0.12 tag and running 
mvn install
You can include any platform module in your mvn build by placing the following in your pom: {% highlight xml %} motech-repo Motech Maven Repository http://nexus.motechproject.org/content/repositories/public org.motechproject [module name] 0.12 {% endhighlight %}

Installation Instructions

With this release we've created debian and rpm packages that can install MOTECH and it's dependencies for you. We'll be posting those shortly. If you want to manually install MOTECH you will need:
  • Tomcat
  • MySQL
  • CouchDB
  • Java
  • ActiveMQ
