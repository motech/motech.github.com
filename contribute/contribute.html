Join the MOTECH Community

MOTECH is an Open Source project so there are many ways you can help us out. You can start simply by contacting us and letting us know you are interested. We specifically need help in some of these areas:

Ways to Contribute

  • Develop
    We need help developing new modules, new telephony adapters, and our web-based UI for administering the system.
  • Document
    We would love to tell our story, perhaps you can help. We need help documenting use cases and other higher level concepts on this website, as well as improving our developer documentation.
  • Test
    We need to make sure our system is robust in order to be used effectively to improve health outcomes. We could use some help testing MOTECH releases.
  • Design
    We have some big plans around making MOTECH extremely simple for end users to administer. We could really use some help firming up these plans and then delivering on them.
