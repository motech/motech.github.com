Develop MOTECH

If you are a software developer and you'd like to start contributing to the MOTECH codebase, here is how to get started:

Step 1: Introduce Yourself

Drop us a line via our contact form. Let us know a bit about yourself, your programming experience, and how you'd like to help. We'll help identify some bugs or work items you can get started on.

Step 2: Join the Discussion

Join the MOTECH Developer mailing list.

Step 3: Learn about MOTECH

Peruse the technical documentation and javadoc.

Try the MOTECH demos.

Check out our development wiki for tips on our development and release process.

Step 4: Configure Your Dev Machine

Configure your machine.

Get an account on Gerrit so that you may review and submit code changes.

Clone the source code and start coding.
