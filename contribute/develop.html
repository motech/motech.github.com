Develop MOTECH

If you are a software developer and you'd like to start contributing to the MOTECH codebase, here is how to get started:

  1. Drop us a line via our contact form. Let us know a bit about yourself, your programming experience, and how you'd like to help. We'll help identify some bugs or work items you can get started on.
  2. Join the MOTECH Developer mailing list.
  3. Configure your development environment.
  4. Check out our development wiki for tips on our development and release process.
  5. Learn about MOTECH. Peruse the technical documentation and try out the MOTECH demos.
