Detailed Description

AtomFeedService allow users to fetch an atom feed provided by the OpenMRS Atom Feed module.


If the atom feed contains new entries, corresponding events will be raised in the Motech server that correspond to those actions.

Clients interested in listening for events should see EventSubjects for a full description of possible events that may be generated by the Atom Feed. The data associated with each event can be accessed with EventDataKeys.

Fetch changes from the OpenMRS Atom Feed since sinceDateTime and lastId.

If sinceDateTime is null, this is equivalent to calling fetchAllOpenMrsChanges()

Parameters:
sinceDateTimeA date time formatted as: 2012-07-09T13:55:35
lastIdThe id of an entry to further filtering on. The id is only relevant it there is an entry that was updated at sinceDateTime, otherwise this parameter is ignored
