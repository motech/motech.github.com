Detailed Description

Represents a time when the OpenMRS Atom Feed was last updated.

There should only ever be one instance saved in the database at a time. The last update is represented as a time, and an id of the last processed entity. In theory, it's possible 2 entities can be updated at the same time, but with different ids.
