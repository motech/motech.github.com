Detailed Description

Contract for tree command.


Known use cases:
Call back from tree node, executed whenever tree node is processed. See also Node#getTreeCommands()
Dynamic prompts by setting command on prompt, result of command will be used in prompt text. See also AudioPrompt#setCommand(ITreeCommand), TextToSpeechPrompt#setCommand(ITreeCommand)
