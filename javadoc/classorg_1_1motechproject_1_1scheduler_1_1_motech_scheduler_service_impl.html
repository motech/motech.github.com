void org.motechproject.scheduler.MotechSchedulerServiceImpl.scheduleJob ( CronSchedulableJob  cronSchedulableJob  ) 

Schedules the given schedulable job.

The Job ID by which the job will be referencing in the future should be provided in an Instance of MotechEvent in SchedulableJob (see MotechEvent.jobId)

If a job with the same job ID as the given exists, this job will be unscheduled and the given schedulable job will be scheduled
