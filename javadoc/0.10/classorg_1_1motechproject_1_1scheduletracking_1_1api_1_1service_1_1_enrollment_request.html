This is the request document to create an enrollment.

This is the constructor which takes in required parameters to create an enrollment.

Parameters:
externalId
scheduleName
preferredAlertTime
referenceDate- This is the start date of the schedule based on which all the window duration calculations are made. In case of enrollment into milestones other than first milestone, this date is not used. Default value is today
referenceTime- This is the start time of the schedule based on which all the window duration calculations are made. In case of enrollment into milestones other than first milestone, this time is not used. Default value is midnight
enrollmentDate- The date of enrollment. In case of enrollment into milestones other than first milestone, this becomes the start date of that milestone. Default value is today
enrollmentTime- The time of enrollment. In case of enrollment into milestones other than first milestone, this becomes the start time of that milestone. Default value is midnight
startingMilestoneName- Name of the milestone to enroll against Default value is first milestone name from the schedule
metadata- This is list of string key value pairs associated with an enrollment, which can be used to store some additional information about the enrollment. Default value is empty list
