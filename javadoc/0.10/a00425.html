OutboundVoiceMessage org.motechproject.outbox.api.service.impl.VoiceOutboxServiceImpl.getNextPendingMessage ( String  partyId)

Returns the next pending message from the outbox of the party identified by the given party ID, or NULL if there is no more pending messages in the outbox.

A next pending message is a not expired message with the status "Pending" that belongs to this party, the oldest creation time and the highest priority among other pending messages in the party outbox.
