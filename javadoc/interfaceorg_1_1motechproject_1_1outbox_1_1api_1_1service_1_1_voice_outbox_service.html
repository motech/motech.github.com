Detailed Description

Voice Outbox Service interface Provides methods to get information and manage messages in the party voice outbox.

Each party (Patient, Nurse, Doctor, etc.) can have a voice outbox.

Igor (iopus.nosp@m.hnye.nosp@m.v@2pa.nosp@m.ths..nosp@m.com)

OutboundVoiceMessage org.motechproject.outbox.api.service.VoiceOutboxService.getNextPendingMessage ( String  partyId)

Returns the next pending message from the outbox of the party identified by the given party ID, or NULL if there is no more pending messages in the outbox.

A next pending message is a not expired message with the status "Pending" that belongs to this party, the oldest creation time and the highest priority among other pending messages in the party outbox.
