void org.motechproject.sms.api.service.SmsService.sendSMS ( String  recipient,
String  message 
) [package]

Raises a SendSMS event for sending an sms to a recipient.

An accompanying listener must be implemented which can deliver the message by interfacing with an sms provider.

Parameters:
recipient- the phone number of the recipient
message- the message text The payload raised with the event has the following values: number - List<String> containing the recipients for the message message - String containing the message text delivery_time - org.joda.DateTime used to schedule the delivery of the message at the specified time

Implemented in org.motechproject.sms.api.service.SmsServiceImpl, and org.motechproject.sms.api.service.SmsServiceImpl.
