At this point in time the Decision Tree, IVR Kookoo and Pill Reminder modules still require documentation. We expect to have those completed by Fri. March 16th.

Alerts

Collects alerts for users in an inbox-like container

Appointments

Provides appointment scheduling and reminding

CMS Lite

Provides basic content storage and retrieval

Decision Tree

Provides APIs for constructing an IVR decision tree

IVR

Provides basic specification for integrating platform with an IVR service provider; Also, connects the platform IVR with an asterisk server using VoiceGlue VXML browser

IVR Kookoo

Integrates the platform with Kookoo's hosted IVR

IVR Voxeo

Integrates the platform with Voxeo's hosted IVR

Message Campaign

Enrolls users in message campaigns with flexible content scheduling rules

MRS (Medical Record System)

Provides basic specification for integrating platform with a medical record system

OpenMRS

Integrates platform with OpenMRS system

Outbox

A voicemail like messaging system for end users

Pill Reminder

A flexible reminder system focused on medication

Schedule Tracking

Enrolls users for alerts based on complex scheduling rules

SMS

Provides basic specification for integrating platform with an SMS provider to send/receive SMS messages

SMS HTTP

Allows platform to integrate with an HTTP-based SMS gateway to send SMS messages

SMS SMPP

Allows platform to integrate with an SMPP-based SMS gateway to send/receive SMS messages
