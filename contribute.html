Help Us!

Motech is an Open Source project so there are many ways you can help us out. You can start simply by contacting us and letting us know you are interested. We specifically need help in some of these areas:

Ways to Contribute

  • Website
    This site can use a lot of work. We'd love to tell our story, perhaps you can help.
  • Documentation
    We've got a lot of code, but not a lot of documentation. Any help here is greatly appreciated
  • Configuration
    Know puppet or chef? We could use help getting some recipes in place to make installation easier.
  • UI/UX
    We've got some big plans around making Motech extremely simple for end users to administer. We could really use some help firming up these plans and then delivering on them.
  • Development
    We've got a long list of tasks that we're working our way through. We need help developing new modules, new telephony adapters and could really use an OSGI expert to help us solidify our module system.
